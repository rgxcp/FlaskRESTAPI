# Python REST API
Simple REST API using Python Flask &amp; MariaDB

## Requirement
1. Python 3.x
2. MariaDB
3. Flask
4. Flask-RESTful
5. PyMySQL

## Database Structure
Table Name : cinemas

Table Structure
1. id --> INT, NOT NULL, AUTO INCREMENT, PRIMARY KEY
2. title --> VARCHAR
3. year --> VARCHAR
4. director --> VARCHAR
